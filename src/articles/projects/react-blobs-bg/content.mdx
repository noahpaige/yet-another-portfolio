`react-blobs-bg` is a small React library for rendering an animated background with with HTML Canvas. It powers the animated background on this portfolio and is designed to be **easy to drop into any React app** while staying **performant and customizable**.

You can find the package on [npm](https://www.npmjs.com/package/@noahpaige/react-blobs-bg) and on [GitHub](https://github.com/noahpaige/react-blobs-bg).

Check out [a live example on CodePen](https://codepen.io/noahpaige/pen/EaKodox) to see it in action.
